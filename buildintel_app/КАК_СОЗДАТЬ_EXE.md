# Как создать .exe файл BuildIntel

## Быстрая инструкция

### Шаг 1: Установите зависимости
```bash
pip install -r requirements.txt
pip install pyinstaller
```

### Шаг 2: Запустите сборку
```bash
.\build.bat
```

### Шаг 3: Найдите готовый файл
После сборки файл будет в папке: `dist\BuildIntel.exe`

## Запуск .exe файла

1. Дважды кликните на `dist\BuildIntel.exe`
2. **Важно:** Перед запуском убедитесь, что бэкенд сервер запущен!

## Почему нужен сервер?

Десктопное приложение работает как **клиент**, который обращается к **серверу** (бэкенд FastAPI):

- Сервер обрабатывает запросы к OpenAI API
- Сервер использует Selenium для парсинга сайтов
- Сервер безопасно хранит API ключи

Без сервера приложение не сможет работать, так как:
- Нет доступа к OpenAI API
- Нет возможности парсить сайты
- Нет обработки данных

## Автозапуск сервера (опционально)

Чтобы не запускать сервер вручную каждый раз:

1. Создайте файл `start_backend.bat` в корне проекта:
```batch
@echo off
cd /d "%~dp0"
venv\Scripts\activate
python -m uvicorn backend.main:app --reload --host 0.0.0.0 --port 8000
```

2. Добавьте его в автозагрузку Windows:
   - Нажмите Win+R
   - Введите `shell:startup`
   - Скопируйте туда ярлык на `start_backend.bat`

Теперь сервер будет запускаться автоматически при загрузке Windows!
